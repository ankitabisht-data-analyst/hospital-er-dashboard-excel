# 📁 Excel Folder

| File | Description |
|---|---|
| `hospital_er_dashboard.xlsx` | Excel workbook containing: raw data loaded via Power Query, pivot tables for ER patient count/wait time/satisfaction score trends, and a final Dashboard sheet with all charts arranged by sequence. |

**How to use:** Open in Microsoft Excel. If prompted, click Enable Content to allow Power Query and pivot table refresh. The Dashboard sheet shows the final output; individual chart sheets (Daily ER No. of Patients, Average Wait Time Daily Trend, Patient Satisfaction Score Daily) contain the underlying pivot charts.
